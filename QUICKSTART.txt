🔍 PORT DETECTION & TROUBLESHOOTING:

If the admin panel is not accessible, the setup script automatically
detects and reports port issues. You can also run the diagnostic tool:

   $ sudo ./diagnose.sh

This will check:
   ✓ Ports 5000 (Flask), 80 (Nginx), 443 (HTTPS)
   ✓ What processes are using those ports
   ✓ Firewall rules and blocked ports
   ✓ Service status
   ✓ Network connectivity
   ✓ Recent error logs

If ports are blocked by firewall, the script provides fix commands:
   $ sudo ufw allow 5000/tcp
   $ sudo ufw allow 80/tcp
   $ sudo ufw reload

⚡ QUICK DIAGNOSTIC COMMANDS:

Check if Flask is running on port 5000:
   $ sudo lsof -i :5000

Check if Nginx is running on port 80:
   $ sudo lsof -i :80

Check all listening ports:
   $ sudo netstat -tuln | grep LISTEN

View PISONET logs:
   $ sudo tail -f /var/log/pisonet/server.log

View Nginx errors:
   $ sudo tail -f /var/log/nginx/error.log

Check service status:
   $ sudo systemctl status pisonet
   $ sudo systemctl status nginx

Restart services:
   $ sudo systemctl restart pisonet
   $ sudo systemctl restart nginx

🎯 COMMON SCENARIOS:

1️⃣  Admin panel loads but can't add credits
   → Check firewall rules: sudo ufw status
   → Restart Flask: sudo systemctl restart pisonet

2️⃣  "Connection refused" error
   → Check if Flask is running: sudo lsof -i :5000
   → Check if Nginx is running: sudo lsof -i :80
   → Run diagnostic: sudo ./diagnose.sh

3️⃣  Can access with localhost but not from another machine
   → Firewall is blocking port 80
   → Fix: sudo ufw allow 80/tcp
   → Reload: sudo ufw reload

4️⃣  Port 5000 or 80 already in use
   → Find what's using it: sudo lsof -i :5000
   → Stop the process or change Flask port in server.py

📱 CLIENT SETUP:

After server is running, setup client machines:

Windows Client:
   1. Install Python 3.8+ from python.org
   2. Download client.py to client machine
   3. Run: python client.py http://<server-ip>:5000

Linux Client:
   1. Install: sudo apt install python3 python3-tk python3-requests
   2. Download client.py
   3. Run: python3 client.py http://<server-ip>:5000

Auto-start on Windows:
   1. Create scheduled task
   2. Or add to startup folder
