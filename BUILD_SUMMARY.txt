🆘 PORT TROUBLESHOOTING QUICK REFERENCE:

Problem: Admin panel won't load
Solution:
   1. Run diagnostic: sudo ./diagnose.sh
   2. Check Flask: sudo systemctl status pisonet
   3. Check Nginx: sudo systemctl status nginx
   4. Restart both: sudo systemctl restart pisonet nginx

Problem: Connection refused error
Solution:
   1. Check ports: sudo lsof -i :5000,:80
   2. Check firewall: sudo ufw status
   3. Open ports: sudo ufw allow 5000/tcp 80/tcp
   4. Reload: sudo ufw reload

Problem: Accessible from server, not from network
Solution:
   1. Firewall blocking port 80
   2. Run: sudo ufw allow 80/tcp
   3. Reload: sudo ufw reload
   4. Test: curl http://<server-ip>/

Problem: Port already in use
Solution:
   1. Find process: sudo lsof -i :5000
   2. Stop process: sudo kill -9 <PID>
   3. Or change port in server.py & setup.sh
